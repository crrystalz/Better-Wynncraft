# Better Wynncraft
Wynncraft is a remarkable MMMORPG built within Minecraft, in the form of a Minecraft Server. It's a great server, but in order to spice up your experience, here is the best modpack that improves your gameplay, while improving the aesthetics, enjoy!

## Download Instructions
**Important: IMPORTING A CLONED ``.zip`` FROM THIS REPOSITORY INTO CURSEFORGE WILL NOT WORK.**

### Step One:
Head on over to the [releases page](https://github.com/crrystalz/Better-Wynncraft/releases), and download the **newest** release, the file should be called "Better.Wynncraft-(version number).zip. This will download a ``.zip`` onto your computer. Do **not** extract this zip file.

### Step Two:
Install the [Curseforge](https://download.curseforge.com) app if you don't already have it. Navigate to the Minecraft page where you should see two headers titled "My Modpacks" and "Browse Modpacks".

### Step Three:
Click on the button called "Create Custom Profile" at the top right of the Minecraft pane. Then click import and select the ``.zip`` file you previously downloaded from the release page of this repostiory. Afterwards, Curseforge will automatically download and install everything for you! :) Once it is done, just hit the play button and you will be all set! Have fun!

If you encounter any problems with this, please create a GitHub issue [here](https://github.com/crrystalz/Better-Wynncraft/issues)
